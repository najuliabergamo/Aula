O que é Java Script?

O JavaScript foi criado com a motivação de tornar as páginas web interativas e dinâmicas. Ele surgiu em 1995 por Brendan Eich em apenas 10 dias, inicialmente para o navegador Netscape.

JavaScript pode ser executado tanto no **navegador** (lado do cliente), onde manipula elementos da página, quanto no **servidor**, usando ambientes como Node.js para criar APIs e sistemas completos.

No navegador, o JavaScript roda dentro de um ambiente isolado chamado **sandbox**, que limita o acesso do script a partes sensíveis do sistema, por segurança.

Existem objetos globais importantes, como:

  • document: permite acessar e manipular o conteúdo da página HTML.
  • console: usado para registrar mensagens de depuração no console do navegador.
  • alert: exibe caixas de mensagem para o usuário.
