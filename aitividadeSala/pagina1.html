Lista com comandos basicos do git

  • git init - cria um repositorio em uma pasta vazia.
  • git clone urlrepositório Clona um repositório
  • git add - adiciona modificacoes do wk no stage
  • git status - mostra o estado atual do repositorio, modificacoes arquivos versionados
  • git commit - cria um commit no repositorio LOCAL
  • git push - envia as modificacoes do repositorio LOCAL para o remoto
  • git log - exibe no terminal os commits do repositorio na branch atual
  • git log --oneline --all - exibe todos os commits de forma resumida
  • git branch - lista as braches do repositorio
  • git checkout - Muda de branch
  • git checkout -b < nome-da-branch > - Cria e muda para uma nova branch
