O css é um arquivo de estilização em cascata criado em 1993, popularizado em 1996, para estilizar e formatar arquivos HTML e o CSS através de cores, estilos, fontes, transições, etc..

O CSS evoluiu co o tempo, surgindo as versões 1 2 e 3

  1. CSS 1 1996 - propriedades básicas de estilização e formatação, controle e posicionamento com display, float e position
  2. CSS 2 1988 - adição de animação e transição, além de melhoria de position
  3. CSS 3 2008 - incremento gradual de novas funcionalidades do css com animação e ferramentas de posicionamento como display: flex (2008) e grid (2013)

Podemos incluir código css de 3 formas diferentes

CSS Inline
Quando definimmos código css de estilização no próprio elemento html como:
<span style="color:red">Texto de cor vermelha com css Inline</span>
CSS Incorporado
Quando definimos código css por meio da tag <style>
CSS Externo
Quando definimos código CSS de estilização em outro arquivo: < Link rel="stylesheet" href="styles.css">
======= CSS

O CSS é um documento de estilização em cascata criado em 1993, com sua primeira versão amplamente utilizadsa em 1997. A linguagem CSS permite definir regras que definem características para os elementos HTML como cor, borda e comportamentos, além de animações e transições

Ao longo dos anos o css evoluiu e hoje temos o CSS 1 2 e 3, cada uma das versões adicionou novos recursos que eram demandados no período.

  1. CSS 1 - propriedades básicas de estilização e formatação, além de controle do posicionamento de elementos html com propriedades float e position
  2. CSS 2 2008 - adição de propriedades de animação e transição.
  3. CSS 3 2008 - incremento gradual de novas funcionlaidades do CSS com animação e ferramentas de posicionamento como display:flex e grid (2013).

Uma regra css pode ser definida com a seguinte sintaxe

                Selector{
                Propriedade1:Valor;
                Propriedade2:Valor;

                PropriedadeN:Valor
                }
